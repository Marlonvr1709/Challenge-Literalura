---CHALLENGE-LITERALURA

Literalura es una aplicación de consola desarrollada en Java con Spring Boot que permite consultar información de libros utilizando la API pública Gutendex y almacenar los resultados en una base de datos PostgreSQL.

La aplicación permite buscar libros por título, registrar la información obtenida desde la API y consultarla posteriormente mediante diferentes opciones desde un menú interactivo en consola.
